## Requirements

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install following packages :-
* Pygame
* OpenCV
* Matplotlib
* Numpy

```bash
pip install pygame
pip install opencv-python
pip install matplotlib
pip install numpy
```
